Fett gedruckter Text mit <bold>

In diesem Text hat es Fett gedruckten Text, der unschwer zu erkennen ist.

Wichtige Informationen mit <Strong>

Wichtiger Text oder wichtige Informationen können mit Semantisch speziell ausgezeichnet werden. Auch wenn Visuell kein Unterschied zum fett gedruckten Text zu erkennen ist , hat er doch eine andere semantische Bedeutung.

Eine ganze Reihe weiterer, semantisch bedeutender Tags

Text kann schräg (italic) ausgezeichnet werden
Text kann betont (emphasized)ausgezeichnet werden.
Text kann markiert (markiert) ausgezeichnet werden.
Text kann kleiner (small) ausgezeichnet weren.
Text kann durchgestrichen (deleted) ausgezeichnet werden
Text kann eingefügt (inserted) ausgezeichnet werden
Text kann tiefgestellt (subscript) ausgezeichnet werden
Text kann hochgestellt (superscript) ausgezeichnet werden
Code kann int c = 1 ausgezeichnet werden
